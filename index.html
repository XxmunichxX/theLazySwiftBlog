Greetings fellow Swifter! Here's my advice for you: be like lazy variables and lazy views in the Swift programming language: do the right work, at the right time, only when it's truly needed, in the name of efficiency.

TIP OF THE DAY FOR THE SAVIOR iOS DEVELOPER: Make sure your app uses the MAIN THREAD only to interact with the user interface. Direct all other operations to a background thread, operation queue, or Grand Central Dispatch queue. Remember that Task {} inherits the priority from its creation context and so, if you are going to use it in the body variable of your SwiftUI View, it will run your code on the main thread automatically. For that reason, to make the “heavy-task” function run on a background thread, make it ASYNC and use AWAIT when calling it or detach the Task from the main thread.
