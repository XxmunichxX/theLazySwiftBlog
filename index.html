TIP OF THE DAY FOR THE SAVIOR iOS DEVELOPER:

Make sure your app uses the MAIN THREAD only to interact with the user interface. Direct all other operations to a background thread, operation queue, or Grand Central Dispatch queue. Remember that Task {} inherits the priority from its creation context and so, if you are going to use it in the body variable of your SwiftUI View, it will run your code on the main thread automatically. For that reason, to make the “heavy-task” function run on a background thread, make it ASYNC and use AWAIT when calling it or detach the Task from the main thread. Go in peace now, spread the word.
